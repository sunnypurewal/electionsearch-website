Election Search

Just news. No ads. No distractions.

A new search engine

that helps us find news articles about the 2019 Canadian Election. It is an open-source project and the code is available on GitHub.

Why not just use Google?

Google, Facebook, and Twitter all make a huge portion of their revenue through selling ads. When we search for election news, we want to see relevant articles from reputable sources and that's it. The ads we see are not only distractions, but are sometimes paid for by illegitimate foreign entities.

Just the news, thanks.

Election Search aggregates articles from all major Canadian media outlets and we try our best to only include reputable sources. If you have any suggestions or comments, please send us an email.
